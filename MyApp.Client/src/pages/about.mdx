A full-stack .NET 10 + React 19 + Vite project template that combines the power of ServiceStack with a fast Vite-powered React development experience, including:

- [React 19](https://react.dev) - Modern React with concurrent features
- [Tailwind CSS 4](https://tailwindcss.com) - Utility-first CSS framework with dark mode support
- [shadcn/ui Components](https://react.servicestack.net) - Beautiful accessible React components
- [ASP.NET Identity Auth](https://learn.microsoft.com/en-us/aspnet/core/security/authentication/identity) - Full Identity integration with Tailwind-styled pages
- [Entity Framework](https://learn.microsoft.com/ef/) & [OrmLite](https://docs.servicestack.net/ormlite/) - Dual ORM support; OrmLite for application data, EF Core for Identity
- [AutoQuery](https://docs.servicestack.net/autoquery/) - Rapidly develop data-driven APIs and CRUD UIs
- [Built-in UIs](https://servicestack.net/auto-ui) - Explore APIs at **/ui/** and manage data at **/admin-ui/**
- [Docker Deployments](https://docs.servicestack.net/kamal-deploy) - GitHub Actions CI/CD with Kamal for zero-downtime deployments
